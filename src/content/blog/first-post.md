このブログをはじめました。ずっとやろうと思っていたのですが、やっと動き出した感じです。

## なぜ今さらブログなのか

SNS で発信するのもいいのですが、長い文章を書ける場所がほしかった。140 文字じゃ足りないことって、意外と多いんですよね。

それに、自分の考えを整理するには文章を書くのが一番だなと、最近改めて思っています。

## Astro を選んだ理由

静的サイトジェネレーターはいくつか試したのですが、Astro にしました。

- Markdown で記事が書ける
- ビルドが速い
- JavaScript を最小限にできる
- Cloudflare Pages との相性がいい

特に「必要な場所にだけ JS を使う」という哲学が好きです。ブログにそんなに重いフレームワークはいらない。
